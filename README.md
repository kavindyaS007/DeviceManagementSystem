# DeviceManagementSystem

Back-end : SpringBoot

Features ;
  1)  Add a device
  2)  View all devices
  3)  View a device by Id
  4)  Update a device
  5)  Delete a device
  
to Run;

>>git clone https://github.com/kavindyaS007/DeviceManagementSystem.git

Create a database with the name of "deviceDb"

Open the "back-end" folder in an IDE and run
